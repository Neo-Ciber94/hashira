# Hashira Templates

Create a project using any of these templates using the CLI: `hashira new`

```bash
Creates a new project

Usage: hashira.exe new [OPTIONS]

Options:
      --name <NAME>            name of the project
  -p, --path <PATH>            Path to create the project
      --actix-web              Use the actix-web template
      --axum                   Use the axum template
      --log-level <LOG_LEVEL>  [default: info]
      --rocket                 Use the rocket template
      --deno                   Use the deno template
      --example <EXAMPLE>      Use one of the examples
      --force                  Allow the template to overwrite existing files in the destination 
      --quiet                  Whether if not emit nothing in console
  -h, --help                   Print help
  -V, --version                Print version
```
