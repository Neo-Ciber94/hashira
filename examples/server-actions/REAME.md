# Hashira x Axum

A hashira template using `Axum`.

## Prerequisites

- Hashira CLI:
  - <https://crates.io/crates/hashira-cli>
    - `cargo install hashira-cli --force`

## Usage

- Download this template
  - `hashira new --axum --name `
- Run the example
  - `hashira dev` **(watch mode)**
  - `hashira run` **(only run)**
