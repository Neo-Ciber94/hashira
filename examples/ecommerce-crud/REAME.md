# Hashira x Axum

A hashira template using `Axum`.

## Prerequisites

- Hashira CLI:
  - <https://crates.io/crates/hashira-cli>
    - `cargo install hashira-cli --force`

## How to run

- 1. Starts in watch mode
  - `hashira dev`
- 2. **Only** build:
  - `hashira build`
- 3. Build and run run:
  - `hashira run`
